**按钮行布局说明（播放控制区）**：
- 顺序播放、随机播放：wrap_content，在 HorizontalScrollView 中，小屏可横滚
- 黄色按钮：`layout_weight=1`，与绿色按钮同行，填充到最右侧；文字过长时走马灯滚动
- 黄色按钮仅在"有活跃播放"或"有上次播放记录+开关开启"时显示

## 播放页

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  ┌───────────────┐   歌曲：夜に駆ける                                        │
│  │               │   艺术家：YOASOBI                                         │
│  │   封面 1:1    │   专辑：THE BOOK                                          │
│  │  （正方形，   │                                                           │
│  │  centerCrop） │   00:42 / 04:19                                           │
│  │               │   ━━━━●━━━━━━━━━━━━━━━━━━━━━━━━━━  ← SeekBar（可聚焦）  │
│  │               │                                                           │
│  │               │   歌词：                                                  │
│  └───────────────┘   昨日までの僕らは                                        │
│  （占宽约 30%）      [高亮粗体] 明日を知らない                               │
│                      また逢う日まで さよなら                                  │
│                      （↑ 自动滚动，当前行高亮白色粗体，其余行灰色）          │
│                      （歌词区 weight=1 撑满剩余高度，无滚动条）              │
│                                                                              │
├──────────────────────────────────────────────────────────────────────────────┤
│ [上一首]  [播放/暂停]  [下一首]  [歌词全屏]  [返回文件页]  [定位（黄色）] │
│ （HorizontalScrollView 包裹，小屏可横滚；所有按钮 nextFocusUp → SeekBar）   │
└──────────────────────────────────────────────────────────────────────────────┘
遥控器：SeekBar 聚焦时 ←→ 快进/快退  OK 播放/暂停  ↑↓ 切换控件焦点
```

**封面 / 信息区布局说明**：
- 上半区水平 LinearLayout，封面 weight=3，信息区 weight=7
- 封面用 `SquareFrameLayout` 保证 1:1 比例，`centerCrop` 裁剪
- 歌词区：`NonFocusableScrollView` + `weight=1`，不可聚焦，不阻断遥控器导航

## 设置页

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│  ┌──────────────┐  ┌──────────────────────────────────────────────────────┐  │
│  │ 显示设置     │  │ 播放设置                                              │  │
│  │ 播放设置  ←选│  │                                                       │  │
│  │ 应用设置     │  │  歌词                                                 │  │
│  │ 其它设置     │  │  ┌──────────────────────────────────────────────────┐ │  │
│  │ 关于         │  │  │ 播放页歌词字号        当前值：20sp              │ │  │
│  └──────────────┘  │  │ 默认值 20sp，可手动输入                          │ │  │
│                    │  └──────────────────────────────────────────────────┘ │  │
│                    │  ┌──────────────────────────────────────────────────┐ │  │
│                    │  │ 全屏歌词字号          当前值：28sp              │ │  │
│                    │  │ 默认值 28sp，可手动输入                          │ │  │
│                    │  └──────────────────────────────────────────────────┘ │  │
│                    │                                                       │  │
│                    │  其它                                                 │  │
│                    │  ┌──────────────────────────────────────────────────┐ │  │
│                    │  │ 记忆上次播放          当前值：已开启             │ │  │
│                    │  │ 记录上次播放的曲目与进度，下次打开可一键继续      │ │  │
│                    │  └──────────────────────────────────────────────────┘ │  │
│                    └──────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────────────────────────┘
```

**设置页结构说明**：
- 左栏：分类列表（显示设置 / 播放设置 / 应用设置 / 其它设置 / 关于）
- 右栏：选中分类的详情，顶部蓝色标题，条目卡片（标题 + 描述 + 右侧当前值）
- 播放设置下分两个子组：**歌词**（播放页字号、全屏字号）、**其它**（记忆上次播放）
- 应用设置：应用休眠保护开关
- 其它设置：清理缓存（显示当前缓存大小 KB/MB，点击清空歌词+封面磁盘缓存）
- 关于：项目描述 + GitHub 链接（带图标）
- 遥控器：← 聚焦左栏  → / OK 进入右栏  Back 退出设置

## 视觉规范

- 背景：`#172554`（深蓝）；全屏歌词背景：`#0B1220`（更深）
- 主文本：`#EAF2FF`；副文本：`#CBD5E1`；分区标题：`#60A5FA`；歌词灰色：`#94A3B8`
- 焦点：背景高亮 `#334155`，文本保持高对比
- 按钮：绿色 `bg_button_green`、深色 `bg_button_dark`、蓝色 `bg_button_primary`、黄色 `bg_button_light_yellow`（文字 `#3A2A00`）
- 字体：MiSans（regular / medium / bold），随系统 DPI 缩放
- 全局缩放：90% / 95% / 100% / 105% / 110% 五档，通过设置页切换
