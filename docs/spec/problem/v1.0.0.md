# 问题记录

## 光标丢失

实际遥控器操作时，文件浏览目录层级切换（进子目录、回上一级）过程中，会出现失去焦点的情况，即遥控器上下不出现光标（选中效果）。模拟器时可以通过点击文件夹解决，但电视不行。需要保证光标显示

建议：每当进入或退出目录，如果目录有文件光标落在第一个文件（返回下一级下面那个），如果没有则落在返回上一级区域。确保页面刷新有光标有保底的位置。

## 播放控制页面

### 1 布局优化

播放控制，回放当前播放移到随机播放右边，改成浅黄色按钮，占据右边全部宽度（间距还是要留），如果文字还是超过我一行在按钮中从左到右循环滚动。


## 播放器页面

### 1 歌词全屏按钮没有

@docs/spec/design.md 中播放页下一首右侧有歌词全屏按钮，需要增加按钮并实现对应功能。

### 2 歌词截断

目前当行歌词过长时，超过长度会显示省略号。需要将逻辑调整为超过长度自动折行。需要保证所有歌词文本均能显示在页面上。

全屏歌词也一样。

### 3 封面不显示

目前歌曲封面不显示，需要排查。

另外补充封面长宽比不是1:1解决方案。拉伸显示，高度适应显示区域，如果左右宽度有溢出封面框忽略即可。

### 4 进度条支持选中

播放器页面进度条需要支持通过遥控器操作选中，选中后按左右可以实现快进快退。

默认按一次快进快退5秒，长按后随长按时间增加（10、30、60、90），90秒封顶。具体时长档位阈值你根据最佳实践经验控制即可。
